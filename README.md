# CDN 优选节点生成工具 (CDN Best IP Tool)

这是一个纯静态的单页应用 (SPA)，用于帮助用户将 Vmess/Vless/Trojan 节点的地址替换为优选 IP，以提升连接速度。

## ✨ 功能特点

- **多协议支持**: 支持 Vmess, Vless, Trojan 链接解析。
- **智能替换**: 自动保留原始域名作为 Host/SNI，只替换连接 IP。
- **批量生成**: 支持一次性输入多个优选 IP，批量生成订阅列表。
- **隐私安全**: 纯前端运行，所有数据处理均在浏览器本地完成，不会上传到服务器。
- **开箱即用**: 预置 Cloudflare/Cloudfront 等常见 CDN 的优选 IP 示例。

## 🚀 部署到 Vercel

本项目完全适配 Vercel，您可以直接一键部署：

1. Fork 本仓库到您的 GitHub。
2. 登录 [Vercel](https://vercel.com/)。
3. 点击 **"Add New..."** -> **"Project"**。
4. 选择刚才 Fork 的仓库。
5. **重要**: 如果本目录不是仓库根目录，请在 **"Root Directory"** 设置中填入 `cdn_tool`（如果此 README 在根目录则忽略此步）。
6. 点击 **"Deploy"**。

## 🛠️ 本地运行

直接双击打开 `index.html` 即可使用，无需任何服务器环境。

## 📝 使用方法

1. **输入节点**: 粘贴您的原始节点链接（支持多行）。
2. **选择/输入 IP**: 选择 CDN 提供商使用内置 IP，或选择"自定义"填入您扫描到的优选 IP。
3. **生成**: 点击生成按钮，右侧将显示替换 IP 后的新节点链接。
4. **导出**: 支持一键复制或下载为 Base64 编码的订阅文件。
